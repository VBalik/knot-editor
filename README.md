# Редактор топологических узлов · 3D-релаксация

Одностраничное веб-приложение: рисуете узел мышью, задаёте проходы над/под, и
физическая релаксация упругой струны раскрывает его в 3D в каноническую форму.
Тривиальная петля (в т.ч. сильно закрученная) всегда распрямляется в **окружность**.

**Живая версия:** открывается по адресу GitHub Pages этого репозитория.

## Возможности
- 2D-редактор: рисование замкнутой кривой, сглаживание (Catmull-Rom), поиск
  самопересечений, назначение над/под кликом (классическая диаграмма узла).
- Примеры узлов: трилистник 3₁, восьмёрка 4₁, пятилистник 5₁, семилистник 7₁.
- 3D-физика (overdamped, без инерции и кручения):
  - сила распрямления по углу между сегментами (энергия изгиба);
  - самоотталкивание прядей (собственная толщина струны);
  - жёсткие сегменты постоянной длины, само-непроникновение по отрезкам;
  - тип узла определяется топологически (определитель диаграммы), поэтому
    скручивание (writhe) свободно раскручивается, а настоящий узел — нет.
- Регулируемые цвет, толщина трубки, сила распрямления и самоотталкивания.
- Экспорт 3D-узла в OBJ.

## Технологии
Один самодостаточный HTML-файл. [three.js](https://threejs.org) r128 с CDN.
Ни сборки, ни сервера, ни зависимостей.

## Как запустить локально
Просто откройте `index.html` двойным кликом в браузере.
