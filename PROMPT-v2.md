# Промпт: Редактор топологических узлов с 3D-релаксацией

## Постановка задачи

Одностраничное веб-приложение `knot_editor.html` — один самодостаточный
HTML-файл (three.js r128 с CDN, без сборки и сервера, открывается двойным
кликом). Пользователь рисует мышью замкнутую кривую — диаграмму узла, задаёт
на пересечениях проходы «над/под», приложение строит по диаграмме
пространственную кривую и физической релаксацией приводит её к равновесной
форме: тривиальный узел распрямляется, настоящий узел принимает гладкую
объёмную форму.

*(Раздел «Физика релаксации» пишется заново — здесь намеренно отсутствует.)*

## Интерфейс

Два окна рядом; все управляющие элементы — в компактных тулбарах в одну строку
над окнами (одинаковая высота слева и справа), сами окна чистые.

### Левое окно — 2D-редактор диаграммы
- Рисование: зажать мышь и вести; для замыкания вернуться в стартовый кружок
  (авто-замыкание). Кривая сглаживается (Catmull-Rom).
- Самопересечения находятся автоматически. Отрисовка классическая: верхняя
  прядь сплошная, нижняя — с небольшим чистым разрывом, целиком внутри серого
  кольца-хендла (разрыв ~полширины линии с каждой стороны, ровные торцы,
  диаметр кольца не увеличивается).
- Клик по кольцу переключает над/под; 3D-вид обновляется сразу.
- Тулбар: «Очистить»; примеры узлов сегмент-кнопками 3₁ 4₁ 5₁ 7₁; чип «×N» —
  число пересечений; чип типа узла — «тривиальный» или «узел · det N»
  (определитель диаграммы); краткая подсказка.

### Правое окно — 3D-вид
- Кривая отображается трубкой (TubeGeometry) с бликами; парит над полом с
  сеткой и отбрасывает мягкую тень; пол всегда чуть ниже нижней точки узла.
- Освещение со всех сторон: узел никогда не тонет в темноте и не «выбеливается»,
  цвет и объём читаются при любом ракурсе.
- Камера орбитальная: вращение мышью, зум колесом; узел всегда целиком в кадре
  и по центру окна (авто-кадрирование, панорама отключена).
- Тулбар: цвет трубки; толщина (⌀) со значением; ползунок «Жёсткость» 0–30;
  ползунок «Отталкивание» 0–20 (значения — числом рядом); кнопка «▶ Физика» /
  «⏸ Пауза»; «OBJ» — экспорт трубки в OBJ-файл; «Телем» — выгрузка телеметрии
  релаксации (JSON с параметрами, вершинами кривой, диаграммой и метриками по
  кадрам) для отладки.
- Статус — плашка в правом нижнем углу 3D-окна с пульсирующим индикатором:
  янтарный во время релаксации, бирюзовый по достижении равновесия
  («равновесие достигнуто»).
- Смена «Жёсткости» или «Отталкивания» после достигнутого равновесия сама
  перезапускает релаксацию до нового равновесия.
- Любая необработанная JS-ошибка показывается на странице красным баннером;
  если CDN three.js недоступен — отдельное понятное сообщение.
